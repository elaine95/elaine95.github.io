Computer Graphic Project

Implemented a basic ray-tracer and rendered compond objects using ray casting and local illumination. Within a group of 2 people, I was responsible for implementing recursive ray tracing and rendered effects of secondary reflections, hard and soft shadows, texture mapping, and anti-aliasing. This project was written in C++, the most challenging part of this project is to use traverse algorithm to implement our ray tracer, which we have successfully implemented using an AABB tree function.

Aggregate Learning Mobile App Prototype

A centralized educational platform for a variety of skills but also personalized for users. This project is a mobile prototype designed within a group of 4 and focusing on delivering a user-friendly learning app for young adults. It was completed within one month and had went through stages from brain storm to user research, to designing the UI, then to various phrases of user testing, then at the end to implement the functional prototype.
